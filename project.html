What is Binary Addition?

Binary addition is one of the binary operations which works similarly to the base 10 decimal system, except that it is a basy 2 system. The binary system consists of only two digits, 1 and 0.

The procedure is the same from right to left as we add digits in each column. Adding each one and carry the more significant digit of the sum if there is one to the next column. Let's Look at an example:

We begin by adding rightmost 1 and 1, which we obtain 10. We write the 0 and we carry the 1 to the Second pair of numbers to the left. On this second one, we get 1. We continue going to the left, each one adding up to 1. On the last pair, the only entry is the 1 that we carried from the previous column so we record it in the answer.

With that we'll now be adding binary with fractions which is the same. Except, we use a radix point in the same role as the decimal point in decimal notation.

What we do, we add the radix points which we align first and apply the same addition process as before. The example above, 1011.01 added to 11.011 produces 1110.101.
